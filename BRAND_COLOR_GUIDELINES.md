## Infographic Color Usage Guidelines

### Background Hierarchy

1. **Primary Background**: Use `Primary 500 (#F54AFC)` for main background
2. **Light Backgrounds**: Use `Primary 100-200 (#FBD2C2, #FB9B9A)` for section dividers
3. **Dark Backgrounds**: Use `Primary 800-900 (#79351A, #30150A)` for contrast sections

### Text Color Strategy

- **Main Text**: `#0F172A` on light backgrounds
- **Text on Dark**: `#FFFFFF` on dark backgrounds (Primary 800-900)
- **Accent Text**: `Secondary 500 (#1F7CE4)` for highlights and emphasis

### Logo Integration

- **Auto Selection**: System chooses best logo color based on background
- **Dark Blue Logo**: Use on light brown backgrounds
- **Primary Color Logo**: Use on neutral/dark backgrounds
- **White Logo**: Use on dark brown backgrounds for contrast

### Success Elements

- **Checkmarks**: `#10B981`
- **Positive Indicators**: `#10B981`
- **Achievement Badges**: `#10B981`
- **Success Messages**: `#10B981`

## Technical Implementation

### Color Accessibility

- All text colors meet WCAG AA contrast requirements
- Primary combinations ensure readability
- Dark text (#0F172A) provides 15:1 contrast on white
- White text provides 7:1 contrast on dark backgrounds

### Brand Consistency Rules

1. **Primary Brown**: Must appear in every infographic
2. **Secondary Blue**: Use for 20-30% of visual elements
3. **Success Green**: Reserve for positive actions/indicators
4. **Neutral Colors**: Ensure readability and hierarchy

### Logo Placement Strategy

- **Corner Placement**: Top-right or bottom-right for maximum impact
- **Background Integration**: 15-20% opacity for subtle branding
- **Contrast Check**: Ensure logo visibility against background
- **Size Guidelines**: 10-15% of infographic width
